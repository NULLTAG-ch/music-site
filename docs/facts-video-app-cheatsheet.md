# Higgsfield-App · Text-to-Video Spickzettel (Facts-Video)

Projekttyp **Text to Video** für alle Clips. Gleiche Settings für alle, nur der
Prompt wechselt. Quelle der Inhalte: `scenes.jsx` / Handoff.

## Einstellungen (für jeden Clip gleich)
- **Projekttyp:** Text to Video
- **Modell:** für alle 8 **dasselbe** wählen (gleicher Look!). Empfehlung: **Kling 3.0 Standard** (Tag „Cinematic", 5★, kein „Premium" → günstiger als Pro). Credit-sparende Alternative: **Seedance 1.5 Pro** (Budget + Quality, 5★).
- **Dauer:** **5 s** (Regler) — credit-sparend und passt für Canvas (3–8 s). Wenn Budget reicht: 8 s für vollere Szenen.
- **Seitenverhältnis:** **9:16** (vertikal) — deckt Reels + Spotify-Canvas ab; 16:9 fürs Website-Video später per Reframe. *(Falls das Website-Video Priorität hat: stattdessen 16:9.)*
- Motiv immer **mittig** halten (wegen späterem Reframe/Crop).
